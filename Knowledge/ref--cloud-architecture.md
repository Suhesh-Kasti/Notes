# Public Cloud
- Amazon AWS, Microsoft Azure and Office 365, Google Cloud Platform and Gapps
# Private Cloud
- Here we own our own data centers or lease servers through a data center(co-location)
# On premises
Servers are on our premises
- We might have our own active directory(AD) server to manage all our users and user accounts
- FTP server
# Hybrid Cloud
- Here we use a combination from any of the above
