# WAN Networks
We need this because it will ensure Quality-of-Service(QOS) ie. the guarantee that the packet will reach the other end within certain time limits, minimal packet loss which the internet cannot provide on its own 
1. Leased lines
   - Private connection between businesses
   - Usually referred to as T1/E1 lines
   - If we are not using it, noone is using the connection
   - Not very flexible, if there are alot of locations they don't scale very well
   - It is expensive due to dedicated lines
2. Packet-switched network connections
   Uses virtual circuits to connect networks together
